# Laravel Project

## Prerequisites

1. Install php
2. Install composer
3. Install Docker Desktop

## VS Code Extensions

To make development experience easy, these extensions are very helpful:

1. PHP Extension Pack by Xdebug
2. PHP by DEVSENSE

## Run server

1. Run Docker Desktop

2. Run server commands
    ```zsh
    cd ./laravel-proj
    npm run dev
    ```
